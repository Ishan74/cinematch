How it works

Three steps to movie night

No more endless scrolling through Netflix together. CineMatch handles the decision.

01
👥
Create a group
Start a session and share a 6-digit code. Friends join instantly — no account needed to start swiping.
02
👆
Everyone swipes
Swipe right if you'd watch it, left if not. Filter by genre, mood, runtime, and streaming services you already have.
03
🎉
It's a match
When everyone likes the same movie, you get a match notification with exactly where to watch it tonight.
